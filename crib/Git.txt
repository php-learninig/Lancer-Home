cd <путь до папки куда установим репозиторий>
git init    Создаем/инициализируем репозиторий
git status  проверка репозитории на изменение файлов
git add <имя файла для добавки в коммит (точка добавит все файлы)>
git commit -m "first commit"    сохранить текущее состояние файлов и добавить комментарий "first commit"
