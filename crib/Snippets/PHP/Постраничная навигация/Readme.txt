g_webi_razbitye_stranicy(st,nachalo_ssilki, kol_vidimyx_stranic, kol_dannix, kol_dannix_na_stran [,name_st_var="st"] [,rewrite=0] [,end_stat=""])

st - обязательный числовой параметр, указывает, какая страница в данный момент активная
nachalo_ssilki - текстовая переменная, в ней прописывается начало ссылки, до ввода переменной номера страницы
kol_vidimyx_stranic - числовой параметр, количество видимый страниц, например <<< 3 4 5 6 >>>, здесь количество страниц 4.
kol_dannix - числовой параметр, общее количество данных.
kol_dannix_na_stran - числовой параметр, количество данных на страницу.
name_st_var - не обязательный параметр, указывает имя переменной, которая будет вставляться в ссылку и будет нести в себе номер страницы.
rewrite - необязательный параметр, нужно ли использовать вариант формирования статической ссылки.
end_stat - если используется статический вариант формирования ссылки, тогда можно указать конец ссылки.

пример динамического формирования

<? print g_webi_razbitye_stranicy(10,"<a href="index.php?asd=tr",9,350,5);?>
пример статического формирования

<? print g_webi_razbitye_stranicy(10,"<a href="asd/",9,350,5,"st",1,".html");?>
